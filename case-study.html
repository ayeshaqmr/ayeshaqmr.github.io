Speak Dine: AI-Assisted Voice Accessibility App

Redesigning the food delivery experience for visually impaired users using Flutter and Gemini AI.

Role

Full-Stack Dev

Context

Final Year Project

Timeline

2024–2025

Tech Stack

Flutter, Gemini, Firebase

The Challenge

Traditional food delivery apps rely heavily on complex visual hierarchies, small touch targets, and nested menus. For visually impaired users, screen readers can often fail to parse poorly labeled UI components, making a simple task like ordering a meal incredibly frustrating.

The Solution

Speak Dine entirely bypasses the visual bottleneck by using a conversational AI interface as the primary mode of interaction.

  • Speech-to-Text (STT) & Text-to-Speech (TTS): Users simply talk to the app ("I'd like a medium pepperoni pizza from Pizza Hut"). The app parses the request and reads back the confirmation.
  • Gemini AI Integration: Used the Gemini API to handle Natural Language Understanding (NLU). The LLM extracts intents (order item, check status, cancel) and maps them to our internal database schema.
  • High-Contrast Fallback UI: For users with low vision (rather than complete blindness), the app provides a highly simplified, ultra-high contrast UI with massive touch targets.
"Accessibility shouldn't be a secondary toggle buried in a settings menu. It should be the foundation of the experience."

Technical Architecture

We chose Flutter to build the client so we could deploy to both iOS and Android from a single codebase. The backend runs on Firebase (Firestore for menus and user profiles, Firebase Auth for seamless login).

// Example: Sending parsed text to Gemini API
Future<String> parseOrderIntent(String userSpokenText) async {
  final prompt = "Extract the food item and restaurant from: '$userSpokenText'";
  final response = await gemini.generateText(prompt);
  return response.text;
}

Outcomes

During user testing with visually impaired volunteers, the average time to successfully place a multi-item order dropped by over 60% compared to using standard apps with a screen reader. The project was highly commended by the faculty for its social impact.
